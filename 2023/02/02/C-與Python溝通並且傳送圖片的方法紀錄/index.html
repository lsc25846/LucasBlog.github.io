C#與Python溝通並且傳送圖片的方法紀錄

發表於 2023-02-02 更新於 2024-01-09 分類於 Csharp

轉眼間進入職場也將近一年了，趁著過年有時間來記錄一下今年在職場中摸索出來的技術吧!

C#與Python溝通的方式

當今AOI(自動光學檢測)領域中似乎普遍都是使用C#來作為軟體開發的程式；伴隨著近年來越來越熱的AI話題，那如果想要在現有AOI機台上加入AI的檢測模式的話，就會需要能夠讓C#與Python溝通的手段了。

而網路上存在的C#與Python溝通的方式最容易查詢到的，那肯定是C#調用Python的四種方法了，筆者也是從此文章中提供的後面兩種方式再進行延伸的應用!

後記

現在許多領域都想要與AI沾上邊，透過AI來解決問題；而AI程式大多選擇使用Python撰寫，所以也擁有最多的資源；想要在原有系統上增加AI功能就勢必需要透過某些手段來讓不同程式進行溝通。其實在查詢資料時還有發現其他作法，像是Tensorflow.net，但是考量到摸熟需要時間以及死線的壓力，誤打誤撞摸出今天講的這些方法。
