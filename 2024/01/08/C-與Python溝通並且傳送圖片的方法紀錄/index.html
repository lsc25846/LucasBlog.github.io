C#與Python溝通並且傳送圖片的方法紀錄

轉眼間進入職場也將近一年了，趁著過年有時間來記錄一下今年在職場中摸索出來的技術吧!

C#與Python溝通的方式

當今AOI(自動光學檢測)領域中普遍使用C#進行軟體開發。隨著AI話題的興起，如果想要在現有AOI機台上加入AI檢測模式，就需要讓C#與Python能夠溝通。這裡介紹了C#調用Python的方法，包括在C#中建立Process，透過命令行進行溝通。

C#與Python傳送圖片的方法

方法一：Shared Memory

首先介紹的是通過共享內存來傳送圖片。這個方法涉及在C#中建立影像並將其存儲在記憶體中，然後Python程式可以從記憶體中讀取這些影像。

方法二：Winsocket

第二種方法是通過Winsocket來傳送影像。這種方法的優點是可以在不同電腦之間傳送影像，但速度可能不如共享記憶體方法快。
